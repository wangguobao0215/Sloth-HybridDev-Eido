## Features

- **Central Nervous System Architecture** — Parent Skill (data pathway) + 4 child Skills (sensory S/A → brain H → motor G), Markdown + SQLite dual-write, local-first
- **Agreement Forge (Skill-S)** — Guides SaaS PMs through creating standardized collaboration agreements with mandatory fallback strategies
- **Experiment Log (Skill-A)** — Templated experiment reports, one-click archiving, Bad Case registry, knowledge linking engine
- **Hybrid View (Skill-H)** — Panoramic dependency graph, health scoring, zombie document detection, smart weekly/monthly reports
- **GTM Architect (Skill-G)** — AI capability sanitization, sales battle cards, release notes auto-assembly
- **Agreement State Machine** — 7 states, 13 transitions, violation detection and renegotiation workflow
- **Document Governance** — Freshness mechanism, decay detection, ownership system, change impact analysis
- **Metrics System** — 6 core metrics (agreement coverage, compliance rate, document freshness, linkage rate, GTM turnaround, zombie count)
- **Open-Source Toolchain** — sqlite-utils + Datasette + remark-lint + phodal/adr + git-cliff + Mermaid CLI + markdownlint, zero custom dependencies
