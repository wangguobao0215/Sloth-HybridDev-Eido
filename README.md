## 功能概览

- **中枢神经系统架构** — 父Skill（数据通路）+ 4个子Skill（感觉神经S/A → 大脑H → 运动神经G），Markdown + SQLite 双写，本地优先
- **协作协议生成器 (Skill-S)** — 引导 SaaS PM 创建标准化协作协议，强制降级策略，自动依赖检查
- **AI实验记录员 (Skill-A)** — 模板化实验报告，一键归档，Bad Case 注册表，知识链接引擎
- **混合视图生成器 (Skill-H)** — 全景依赖图、健康度评分、僵尸文档检测、智能周报/月报
- **GTM知识架构师 (Skill-G)** — AI能力脱敏翻译、销售作战卡、发版说明自动组装
- **协作协议状态机** — 7 状态 13 转换，含违约检测与重新协商流程
- **文档治理** — 鲜度机制、腐烂检测、Owner 制度、变更影响分析
- **度量体系** — 6 大核心指标（协议覆盖率/合规率/文档鲜度/关联率/GTM周期/僵尸数）
- **开源工具链集成** — sqlite-utils + Datasette + remark-lint + phodal/adr + git-cliff + Mermaid CLI + markdownlint，零自研依赖
